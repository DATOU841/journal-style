# journal-style skill 公开介绍

## 一句话说明

journal-style 是文衡体系中用于期刊风格分析与投稿辅助决策的工具，基于题录数据、公开信息和已交接的全文样本，提取期刊多维特征并生成可解释的辅助评分，为投稿决策和选题规划提供证据支撑。

## 当前版本状态

- 版本：0.1.6
- 发布日期：以 Git tag / release 记录为准
- 定位：证据约束下的期刊分析与交接版本
- 上游依赖：真实检索、PDF 获取、RAG 入库由检索入库技能承担，journal-style 接收已完成的题录、Zotero 导出文件和全文样本交接物

## 适用场景

- 确定目标期刊前，需要系统了解期刊选题范围、作者构成、参考文献偏好
- 已有初稿或选题方向，需要评估与目标期刊的匹配度
- 需要对比多个同类期刊的风格差异，辅助投稿决策
- 需要追踪期刊选题趋势，规划长期研究方向
- 需要了解期刊的基金要求、机构网络特征
- 需要提取期刊的参考文献生态，为文献综述或引用策略提供参考
- 需要为下游技能提供结构化的期刊风格参考性上下文

## 核心能力

### 期刊身份确认

通过刊号（CN/ISSN）、刊名、主办单位、官网、数据库收录状态建立期刊身份档案。所有后续分析均基于明确的期刊身份，避免同名期刊混淆。

### 题录接收、字段规范、缺口标记和补检索交接框架

journal-style 接收检索入库已完成的题录交接物（Zotero 导出文件、数据库导出文件、题录表）。对接收的题录执行字段规范化检查，标记缺失字段（题名、作者、机构、摘要、关键词、基金、参考文献、发表年份），生成缺口清单与补检索交接物。

题录的真实检索、数据库查询、Zotero 条目保存由检索入库承担。journal-style 不执行数据库实检、不主动获取题录、不主动操作 Zotero。

### 选题专项库框架

在接收的题录基础上，按主题、时间、作者类型、基金类型、机构类型筛选子集，构建专项库。专项库用于聚焦分析特定维度的期刊特征。

### 上游交接缺失时的降级行为

当检索入库未提供题录交接物、Zotero 状态、PDF 状态或 RAG 状态时，journal-style 进入降级模式：

- 只输出已证实的期刊身份信息和公开可获取的投稿运营信息
- 生成缺口清单，明确列出无法完成的分析维度
- 提供补检索建议、补 PDF 建议、补 RAG 建议，包含具体数据需求和优先级
- 不补造结论，不假设数据完整性

### 数据性量化分析报告

基于题录数据，输出以下量化指标：

- 年度发文量、月度发文量
- 基金占比（国家社科基金、省部级基金、青年基金、无基金）
- 作者机构分布（高校、科研院所、政府部门、企业、其他）
- 参考文献统计（平均引用量、中外文献比例、文献时效性分布）
- 合作模式分布（独著、双人合作、三人及以上合作）

所有量化指标均标注数据来源（数据库名称、题录数量、采集时间）。

### 投稿运营与公开声誉证据分析

从期刊官网、数据库页面、社交媒体账号、公开评价平台提取投稿运营信息：

- 投稿邮箱、投稿系统链接、编辑部联系电话
- 公开栏目设置（常设栏目、专题栏目、特邀栏目）
- 投稿要求（字数、格式、匿名审稿、基金要求）
- 公开评价（北大核心、CSSCI、AMI、影响因子、复合影响因子）
- 社交媒体账号（微信公众号、微博、知乎、豆瓣）

所有证据均标注来源 URL 和提取时间。

### 基金与选题关联分析

交叉分析基金类型与选题主题的关联模式：

- 国家社科基金项目倾向的选题类型
- 省部级基金项目倾向的选题类型
- 青年基金项目倾向的选题类型
- 无基金项目倾向的选题类型

输出交叉分布表和关联强度指标。

### 题名风格分析

提取题名的形式特征：

- 题名长度分布（字数分布、中位数、四分位数）
- 高频关键词（题名中出现频次前 20 的词汇）
- 疑问句/陈述句占比
- 副标题使用率

### 历史题录中的主题分布观察

基于题录时间序列，观察历史题录中的主题分布：

- 近三年高频主题词（基于题名和关键词的词频统计）
- 新兴主题词（近一年出现频次显著上升的词汇）
- 消退主题词（近一年出现频次显著下降的词汇）
- 主题词共现网络（高频主题词的共现关系）

**重要说明**：历史题录反映的是已发表内容，不等于编辑部当前征稿偏好。主题分布观察可能滞后于期刊最新导向。

### 材料、方法、论证风格初步观察

基于摘要和关键词，提取方法论特征：

- 高频方法词（问卷调查、访谈、文献综述、案例分析、实证研究、比较研究、历史研究）
- 材料类型分布（档案材料、田野材料、文本材料、统计数据）
- 论证风格标记（理论建构、经验描述、批判反思、政策建议）

**证据等级：初步观察**。基于题名与摘要的方法论特征统计只能支持初步观察，无法确认实际论证结构。若需更深分析，需检索入库提供足够 PDF/RAG 全文样本。

### 样本内结构观察（需上游已提供可召回全文样本）

仅在上游检索入库已完成 PDF/RAG 交接并有可召回样本时，journal-style 可在样本范围内进行结构观察和证据摘录：

- 段落结构分布（引言-文献综述-方法-结果-讨论-结论的结构变体在样本中的分布）
- 引用位置样本摘录（从样本中摘录典型引用位置示例）
- 脚注使用样本摘录（从样本中摘录脚注数量、类型、位置示例）
- 论证链路样本摘录（从样本中摘录理论框架-材料-分析-结论的典型示例）

**重要说明**：journal-style 不承诺自动抽取完整段落组织、引用位置、脚注模式或论证链路。样本内观察仅在已提供样本范围内进行，结果为示例性摘录和分布统计，不构成完整期刊风格提取。所有样本内观察均标注样本量（单篇、10 篇、100 篇）与证据等级。样本量不足 10 篇时标注为探索性观察，10-50 篇标注为中等置信，50 篇以上标注为高置信。

### 参考文献生态分析

基于题录中的参考文献字段，提取参考文献生态：

- 高频被引期刊（被引频次前 20 的期刊）
- 高频被引作者（被引频次前 20 的作者）
- 中外文献比例
- 文献时效性分布（引用文献的发表年份分布）
- 文献类型分布（期刊论文、专著、学位论文、会议论文、报纸文章）

若参考文献数据不完整，标注缺失比例和置信度。

### 作者机构网络分析

基于题录中的作者机构字段，提取作者机构网络：

- 核心作者机构（发文量前 20 的机构）
- 合作网络（机构间合作关系、合作强度、合作社群）
- 地域分布（省份分布、城市分布）
- 机构类型占比（高校、科研院所、政府部门、企业、其他）

### 作者公共身份独立观察

基于题录中公开可见的作者字段和基金字段，提取作者身份分布统计，作为独立观察结果：

- 公开可见的作者身份类型分布
- 基金署名模式（通讯作者基金、第一作者基金、共同基金）

**重要说明**：

- 题录数据通常不包含完整职称字段。作者身份识别仅基于公开可见字段（机构名称中的"博士研究生""硕士研究生""讲师""副教授""教授"等关键词）和用户提供的补充材料
- 无法确认时标注为"待核验"并标记置信度：低置信（仅基于机构关键词）、中置信（基于多个公开字段交叉验证）、高置信（基于用户提供的补充材料）
- 作者身份分析**不进入默认投稿匹配度评分**，仅作为独立观察维度输出
- 若任务中明确启用作者身份匹配度评分，需在任务中声明权重，并在评分报告中标注为弱证据

### 多期刊作者身份对比

若用户提供多个期刊的题录数据，可对比同一作者在不同期刊的发文模式：

- 作者在不同期刊的发文量分布
- 作者在不同期刊的选题类型分布
- 作者在不同期刊的合作模式分布
- 作者在不同期刊的基金类型分布

### 参考文献网络分析

基于题录中的参考文献字段，构建期刊间引用网络：

- 期刊间引用关系（期刊 A 引用期刊 B 的频次）
- 核心引用社群（高频互引的期刊群组）
- 引用时滞（引用文献的发表年份与本刊发表年份的差值分布）

### 投稿匹配度可解释辅助评分

基于用户提供的稿件信息（题名、摘要、关键词、参考文献），计算与目标期刊的可解释辅助评分：

**默认评分维度**（基于可核验字段）：

- 选题匹配度（稿件题名、关键词与期刊高频主题词的相关性）
- 参考文献匹配度（稿件参考文献与期刊高频被引文献的重合度）
- 摘要方法匹配度（稿件摘要中的方法词与期刊高频方法词的相关性）

**可选评分维度**（需在任务中明确启用并声明权重）：

- 作者身份匹配度（稿件作者身份与期刊作者身份分布的契合度，标注为弱证据）
- 基金匹配度（稿件基金类型与期刊基金类型分布的契合度）

**评分输出说明**：

- 默认权重示例：选题匹配度 40%、参考文献匹配度 35%、摘要方法匹配度 25%
- 权重可在任务中配置，所有权重配置在评分报告中明确说明
- 证据不足时给出评分区间（如 60-75 分）或暂缓评分，并在报告中标注证据缺口
- 阈值标准说明（高匹配 >75、中等匹配 50-75、低匹配 <50）在报告中明确说明
- 匹配度评分为辅助决策工具，不能替代编辑部真实偏好

### 需补证方向与风险提示

基于历史题录中的主题分布观察、投稿匹配度评分和参考文献生态，输出需补证方向与风险提示：

- 历史题录中频繁出现的主题类型
- 历史题录中覆盖较少但相关度高的主题
- 跨学科主题在期刊参考文献生态中的分布
- 数据缺口（因题录不完整或样本不足无法确认的风险点）

**风险提示**：

- 历史题录反映的是已发表内容，不等于编辑部当前征稿偏好
- 主题分布观察可能滞后于期刊最新导向
- 需补证方向为研究规划参考，不构成投稿建议

### 文衡状态输出

所有分析报告均包含文衡状态输出：

- 数据来源（数据库名称、题录数量、采集时间）
- 证据链（分析结论对应的原始数据位置）
- 置信度标记（数据完整性、分析可靠性的自评级）
- 未覆盖维度（因数据缺失或技术限制未能分析的维度）

### 下游交接

journal-style 按照契约文档输出结构化上下文，作为下游技能的参考性上下文：

- 向下游传递：期刊选题偏好、题名风格、参考文献生态、样本内结构观察摘录（若有样本）
- 交接内容为参考性上下文，不要求下游严格复现
- 不向下游传递期刊表达腔调或语气模式

下游交接字段结构按照契约文档定义，不输出未定义字段。

## 与其他 skill 的边界

### journal-style 不负责的内容

- 不执行数据库实检、不主动检索题录（需检索入库提供题录交接物）
- 不执行 Zotero 条目保存、不主动操作 Zotero（需检索入库提供 Zotero 导出文件）
- 不执行文献获取、不主动下载 PDF（需检索入库提供 PDF/RAG 交接状态证据）
- 不执行 RAG 入库操作（需检索入库完成入库后提供 RAG 状态证据）
- 不承诺自动抽取完整段落组织、引用位置、脚注模式或论证链路
- 匹配度评分与需补证方向为辅助决策工具，不能替代编辑部真实偏好
- 不对作者未公开的身份信息做推断
- 不向下游传递期刊表达腔调或要求下游复现目标期刊表达腔调

### journal-style 与其他技能的协作

- jiansuo-ruku：jiansuo-ruku 负责从数据库检索题录、导入 Zotero、获取 PDF、执行 RAG 入库，journal-style 负责接收题录交接物、标记缺口、生成补检索交接物并分析
- zhengwen-xiezuo：journal-style 提供期刊风格参考性上下文，zhengwen-xiezuo 自主决定如何使用
- reference-footnote：journal-style 提供参考文献生态和样本内引用观察（若有样本），reference-footnote 负责脚注和参考文献的格式化
- article-polish：journal-style 提供语言风格标记（若有样本），article-polish 负责语言加工

## 输出内容

### 期刊身份确认报告

包含：

- 刊名（中文、英文）
- 刊号（CN、ISSN）
- 主办单位
- 编辑部地址、电话、邮箱
- 官网链接
- 数据库收录状态（CNKI、WoS、维普、万方、北大核心、CSSCI）
- 公开评价（影响因子、复合影响因子、AMI 综合评价）

### 数据性量化分析报告

包含：

- 年度发文量折线图
- 月度发文量柱状图
- 基金占比饼图
- 作者机构分布柱状图
- 参考文献统计表（平均引用量、中外文献比例、文献时效性分布）
- 合作模式分布饼图

所有图表均标注数据来源和题录数量。

### 投稿运营与公开声誉证据报告

包含：

- 投稿邮箱、投稿系统链接、编辑部联系电话
- 公开栏目设置列表
- 投稿要求清单
- 公开评价列表（含来源链接）
- 社交媒体账号列表（含链接）

所有证据均标注提取时间和来源 URL。

### 多维特征分析报告

包含：

- 基金与选题关联交叉表
- 题名风格统计表（长度分布、高频关键词、疑问句占比、副标题使用率）
- 历史题录中的主题分布词云图（高频主题词、新兴主题词、消退主题词）
- 方法风格初步观察表（高频方法词、材料类型分布、论证风格标记，**证据等级：初步观察**）
- 参考文献生态表（高频被引期刊、高频被引作者、中外文献比例、文献时效性分布）
- 作者机构网络图（核心机构、合作网络、地域分布、机构类型占比）
- 作者公共身份独立观察表（基于公开可见字段，标注低/中/高置信度，不进入默认评分）
- 参考文献网络图（期刊间引用关系、核心引用社群、引用时滞）

### 样本内结构观察报告

包含：

- 样本量说明（单篇、10 篇、100 篇）
- 证据等级标注（探索性观察、中等置信、高置信）
- 段落结构分布统计表
- 引用位置样本摘录（典型引用位置示例）
- 脚注使用样本摘录（脚注数量、类型、位置示例）
- 论证链路样本摘录（理论-材料-分析-结论典型示例）

**重要说明**：样本内观察为示例性摘录和分布统计，不构成完整期刊风格提取。仅在上游已交接 PDF 或 RAG 且有可召回样本时输出。

### 投稿匹配度可解释辅助评分报告

包含：

- 综合匹配度评分（0-100 分，或评分区间，如 60-75 分）
- 默认维度评分（选题匹配度、参考文献匹配度、摘要方法匹配度）
- 可选维度评分（若任务中启用：作者身份匹配度、基金匹配度，标注为弱证据）
- 评分模型说明（默认权重与任务中配置的权重）
- 阈值标准说明（高匹配 >75、中等匹配 50-75、低匹配 <50）
- 证据缺口标注（各维度的证据完整性与置信度）
- 匹配度不足的维度说明

仅在用户提供稿件信息时输出。评分为辅助决策工具，不能替代编辑部真实偏好。

### 需补证方向与风险提示报告

包含：

- 历史题录中频繁出现的主题类型列表（含代表性题录）
- 历史题录中覆盖较少但相关度高的主题列表（含相关题录）
- 跨学科主题在期刊参考文献生态中的分布列表（含相关题录）
- 数据缺口清单（因题录不完整或样本不足无法确认的风险点）

**风险提示**：历史题录反映的是已发表内容，不等于编辑部当前征稿偏好。主题分布观察可能滞后于期刊最新导向。需补证方向为研究规划参考，不构成投稿建议。

### 文衡状态文件

按照契约文档定义的稳定字段输出，包含：

- 数据来源标注
- 证据链标注
- 置信度自评级
- 未覆盖维度清单

### 下游交接文件

按照契约文档输出参考性上下文，包含：

- 期刊选题偏好
- 题名风格
- 参考文献生态
- 样本内结构观察摘录（若有样本）

## 设计原则

journal-style 的设计遵循以下原则：

- **证据约束**：所有分析结论均基于可验证的题录数据、公开信息或已交接的全文样本，不补造结论
- **分层置信**：对不同证据来源和样本量标注置信度等级（探索性观察、中等置信、高置信）
- **边界明确**：明确区分可完成的分析维度与因数据缺失无法完成的维度，生成缺口清单与补检索建议
- **辅助定位**：投稿匹配度评分与需补证方向为辅助决策工具，不能替代编辑部真实偏好
- **参考性交接**：向下游提供参考性上下文，不要求下游严格复现
- **不传递腔调**：不向下游传递期刊表达腔调或语气模式，不要求下游复现目标期刊表达腔调
