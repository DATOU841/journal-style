# 作者公共身份与署名基金结构分析协议

## 定位

作者公共身份与署名基金结构分析用于判断目标期刊刊发文章的作者身份结构、署名顺序、学生作者参与度、基金支撑情况和公开可证的师生组合线索。它只做公开元数据层的期刊画像，不推断私人师承关系、内部审稿关系、录用倾向或学术圈层。

## 输入

优先使用题录库、官网目录、PDF 作者简介页或数据库元数据中的公开字段：

- `title` / `题名`
- `year` / `年份`
- `author` / `authors` / `作者`
- `institution` / `affiliation` / `单位`
- `academic_title` / `author_titles` / `职称`
- `degree` / `author_degrees` / `学位`
- `author_stage` / `author_roles` / `作者身份`
- `corresponding_author` / `通讯作者`
- `fund` / `funding` / `基金`
- `advisor_student_relation` / `师生关系说明`

多作者字段按作者顺序用 `;`、`；`、`,`、`，`、`、` 分隔。若某项无法与作者一一对齐，必须标注字段覆盖不足，不能强行补齐。

## 指标

- 作者字段覆盖率、职称字段覆盖率、学位/身份字段覆盖率、基金字段覆盖率。
- 单作者、双作者、三人及以上作者比例。
- 第一作者职称分布：教授、副教授、讲师、研究员、学生、其他、未知。
- 第二作者身份分布：教授/副教授、博士生、硕士生、其他、未知。
- 学生作者参与比例：任一作者字段公开标注为博士生、硕士生、研究生、本科生的题录比例。
- 高年资一作 + 学生二作线索比例。
- 教授/副教授一作 + 硕士二作线索比例。
- 教授/副教授一作 + 博士二作线索比例。
- 明确师生关系说明比例：仅统计 `advisor_student_relation` 或 PDF/官网作者简介中明确写出的关系。
- 通讯作者标注比例和通讯作者位置分布。
- 基金论文比例、基金层级分布：国家级、省部级、市厅级、校级、其他、未知。

## 判断规则

- 样本低于 50 条：只能写“初步观察”。
- 职称字段覆盖率低于 50%：不得给作者职称结构强结论。
- 学位/身份字段覆盖率低于 50%：不得给学生作者参与度强结论。
- 基金字段覆盖率低于 50%：基金结构只能写“题录层初步统计”。
- “导一硕二”不得作为确定关系直接写出，除非公开材料明确标注导师/学生关系；否则只能写“高年资一作 + 硕士二作线索”。
- 不得用作者排位、职称或基金信息推断录用倾向、内部关系、私人师承或投稿捷径。

## 输出

- `03-analysis/journal-author-profile-and-byline-report.md`
- `03-analysis/journal-author-profile-and-byline-statistics.json`

报告必须包含：

- 样本范围和字段覆盖率。
- 作者数与署名顺序结构。
- 第一作者职称分布。
- 第二作者身份分布。
- 学生作者参与情况。
- 高年资一作 + 学生二作线索。
- 明确师生关系说明统计。
- 通讯作者标注情况。
- 基金论文比例和基金层级分布。
- 证据强度、降级提示和不得推断项。
