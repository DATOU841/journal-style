# 数据性量化分析协议

## 1. 定位

`journal-quantitative-analysis-report.md` 是期刊画像前的质量基线报告，用于判断后续题名、趋势、材料、论证和参考文献分析能否给出高置信结论。

## 2. 输出文件

- `03-analysis/journal-quantitative-analysis-report.md`
- `03-analysis/journal-quantitative-statistics.json`

## 3. 覆盖率指标

- 题录覆盖率：采集题录数 / 期刊实际发文总数。
- 年度覆盖率：已覆盖年份数 / 目标年份数。
- 栏目覆盖率：已覆盖栏目数 / 期刊实际栏目数。
- PDF 覆盖率：PDF 数 / 题录总数。
- RAG 可用率：RAG 文档数 / PDF 数。

## 4. 代表性指标

- 年度分布均衡度：各年份样本数标准差 / 均值。
- 栏目分布均衡度：各栏目样本数标准差 / 均值。
- 作者集中度：高频作者占比。
- 机构集中度：机构 HHI 指数。

## 5. 趋势置信度

- 年度覆盖率不低于 80% 才能给中高置信趋势判断。
- 3 年滑动窗口连续增长可标“上升”。
- 近 2 年明显低于前 3 年可标“退潮”。
- 同类关键词近 3 年高频重复可标“拥挤”。

## 6. 数据质量指标

- 重复条目率：重复题名数 / 总题录数。
- 缺失字段率：摘要、关键词、作者、机构等关键字段缺失比例。
- 异常值数量：过短题名、过长题名、异常年份。
- 数据来源冲突数：不同来源之间无法对齐的条目数。

## 7. 质量等级

- `high`：题录覆盖率不低于 80%，PDF 覆盖率不低于 50%，RAG 可用率不低于 50%。
- `medium`：题录覆盖率不低于 60%，且关键字段缺失率不高于 30%。
- `low`：题录覆盖率低于 60%，或 PDF/RAG 严重不足。
