# 标题层初筛协议

## 定位

标题层初筛用于从目标期刊近三年全量题录中剔除明显不相关文献。它只处理题录元数据，不删除 Zotero receipt、PDF、RAG 记录或历史 evidence。

## 保留规则

默认保留：

- 艺术学
- 艺术史
- 艺术理论
- 艺术教育
- 与用户选题材料、方法、时代、概念有明显关联的文章

## 可剔除规则

可剔除：

- 明显不同艺术门类且与用户题目无关
- 建筑、景观、产品、工艺、纯设计实践等明显偏离主题文章
- 纯作品图版或单幅作品标题
- 征稿启事、目录、会议综述、书评等非研究论文

## Ledger

每条剔除必须记录：

```json
{
  "title": "",
  "partition": "",
  "reason": "",
  "had_success_receipt": false,
  "action": "exclude_from_analysis_only"
}
```

禁止静默删除。已有 receipt/PDF/RAG 的条目只能从后续分析分母中排除，不能删除原始证据。
