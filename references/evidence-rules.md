# 证据规则

## 1. 来源层级

1. 官方来源
2. 数据库元数据
3. 期刊官网目录 / 投稿须知
4. RAG / PDF 全文
5. 网络新闻 / 公众号 / 转载页

## 2. 结论门槛

- 题名风格：可主要基于题录。
- 选题趋势：题录 + 摘要 + 关键词优先。
- 材料偏好：题录只能做初判，最好看样本全文。
- 方法偏好：摘要可辅助，全文更稳。
- 论证风格：必须看全文或 RAG。
- 参考文献生态：最好看全文或可解析参考文献。
- 格式体例：优先看投稿须知和全文样本。

## 3. 书写规则

- 不足证据一律标“待补证据”“推测”或“部分支持”。
- 不把题名印象写成全文判断。
- 不把公众号内容写成定论。
- 有覆盖范围时必须写覆盖范围，不要假装全量。
