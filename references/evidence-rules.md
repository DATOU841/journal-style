# 证据规则

## 1. 来源层级

1. 官方来源
2. 数据库元数据
3. 期刊官网目录 / 投稿须知
4. PDF / RAG 全文
5. 网络新闻 / 公众号 / 转载页

## 2. 证据要求矩阵

| 分析维度 | 只需题录 | 需题录+摘要 | 必须全文/RAG | 最低样本要求 | 覆盖率门禁 |
|---|---|---|---|---|---|
| 题名风格 | 是 | 否 | 否 | 50 条题录 | 无；低于 50 条标“样本不足” |
| 选题趋势 | 否 | 是 | 否 | 100 条题录，且覆盖目标年份 | 年度覆盖率不低于 80% |
| 栏目结构 | 是 | 否 | 否 | 覆盖目标年份内全部期次或主要期次 | 年度覆盖率不低于 80% |
| 材料偏好 | 可初判 | 可初判 | 需全文验证 | 20 篇全文样本 | PDF 覆盖率不低于 30% |
| 方法偏好 | 否 | 可初判 | 需全文验证 | 20 篇全文样本 | PDF 覆盖率不低于 30% |
| 论证风格 | 否 | 否 | 是 | 30 篇全文 / RAG 样本 | RAG 可用率不低于 50% |
| 参考文献生态 | 可初判 | 否 | 需全文或参考文献解析 | 30 篇全文或参考文献表 | RAG 可用率不低于 50% |
| 作者机构网络 | 是 | 否 | 否 | 50 条含作者/机构题录 | 作者、机构字段缺失率均不高于 30% |
| 参考文献网络 | 否 | 否 | 需全文或参考文献解析 | 20 篇来源文章且 200 条参考文献记录 | 被引题名、被引年份缺失率均不高于 30% |
| 格式体例 | 否 | 否 | 投稿须知或 10 篇全文 | 10 篇全文样本 | PDF 覆盖率不低于 20% |

## 3. 降级策略

- 题录数低于 50 条：题名风格只能写“初步观察”。
- 年度覆盖率低于 80%：选题趋势必须标“时间跨度不完整”。
- PDF 覆盖率低于 30%：材料、方法和格式分析只能保留题录/摘要层判断。
- RAG 可用率低于 50%：论证风格不得写成完成分析，只能写“待全文证据补足”。
- 参考文献解析样本低于 30 篇：参考文献生态不得给高置信结论。
- 作者或机构字段缺失率高于 30%：作者机构网络不得推断作者圈层、机构生态或录用倾向，只能写公开题录层初步统计。
- 来源文章低于 20 篇或参考文献记录低于 200 条：参考文献网络不得给强结论；被引题名/年份缺失率高于 30% 时，共引和年代结构必须标“待补解析”。

## 4. 证据强度

- `strong`：满足最低样本要求和覆盖率门禁，且有数据库/官方/PDF/RAG 证据。
- `medium`：数据基本可用，但覆盖率或样本数有一项不足。
- `weak`：只有题录、摘要、网页转述或样本明显不足。

## 5. 写法规则

- 不足证据一律标“待补证据”“推测”或“部分支持”。
- 不把题名印象写成全文判断。
- 不把公众号内容写成定论。
- 有覆盖范围时必须写覆盖范围，不要假装全量。
- 低置信结论必须同步写入量化报告和文衡状态。
