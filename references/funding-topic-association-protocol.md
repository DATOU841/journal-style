# 基金与选题关联分析协议

## 定位

基金与选题关联分析用于判断目标期刊中基金论文与选题方向、栏目、关键词、材料类型和方法类型之间的关系。它只做期刊画像和投稿风险辅助，不推断基金项目必然提高录用率，不把未获基金支持的选题写成不适合投稿。

## 输入

优先使用全量题录库、选题专项库或 RAG/全文样本交接中的公开字段：

- `title` / `题名`
- `year` / `年份`
- `column` / `栏目`
- `keywords` / `关键词`
- `abstract` / `摘要`
- `fund` / `funding` / `基金`
- `fund_level` / `基金层级`
- `material_type` / `材料类型`
- `method_type` / `方法类型`

基金字段不足时，不得补猜基金层级；只能按公开字段标注为 `无/未标注` 或 `未知`。

## 指标

- 基金字段覆盖率。
- 基金论文比例。
- 基金层级分布：国家级/重大、省部级、市厅级、校级/高校、其他、无/未标注。
- 各基金层级的高频关键词。
- 各基金层级的栏目分布。
- 各基金层级的材料类型分布。
- 各基金层级的方法类型分布。
- 基金论文与非基金论文的关键词差异。
- 用户关键词在基金论文中的命中情况。

## 判断规则

- 样本低于 50 条：只能写“初步观察”。
- 基金字段覆盖率低于 50%：不得给基金偏好强结论。
- 某基金层级样本低于 5 条：该层级只能列线索，不得写成稳定偏好。
- 不得写“有基金更容易录用”或“无基金不适合投稿”。
- 只能写“该刊已刊样本中，某类基金论文更常见于某些栏目/关键词/材料方法”。

## 输出

- `03-analysis/journal-funding-topic-association-report.md`
- `03-analysis/journal-funding-topic-association-statistics.json`

报告必须包含：

- 样本范围、基金字段覆盖率和证据强度。
- 基金论文比例和基金层级分布。
- 基金层级与关键词、栏目、材料、方法的交叉统计。
- 基金论文与非基金论文的关键词差异。
- 用户关键词命中情况。
- 降级提示和不得推断项。
