# 选题建议规则

## 1. 三类建议

- 高适配选题
- 可补强选题
- 不建议选题

## 2. 推导规则

### 2.1 高适配

同时满足：

- 用户方向处于目标期刊近 3 年热点或稳定方向。
- 材料和方法与该刊常见路径匹配。
- 题名结构接近该刊常见结构。
- 参考文献生态可补齐。

### 2.2 可补强

满足一部分条件，但存在可修复缺口：

- 方向在稳定区域，但材料不足。
- 方向可投，但参考文献生态未接上。
- 题名结构可调整，但不需要重构选题。

### 2.3 不建议

满足任一条件：

- 方向处于拥挤区域。
- 方向处于退潮区域。
- 方向长期边缘。
- 材料方法与期刊明显不匹配。

## 3. 拥挤度判断

- 拥挤：同类关键词近 3 年出现不少于 10 次。
- 稳定：同类关键词近 5 年每年出现 1-3 次。
- 退潮：同类关键词前 3 年出现不少于 5 次，近 2 年少于 2 次。
- 边缘：同类关键词近 5 年出现少于 3 次。

## 4. 题名结构建议

不要直接生成正式题名。只给结构模式：

- 对象 + 问题动作 + 材料
- 概念 + 时代 + 地区 + 方法
- 材料 + 接受 / 传播 / 评价变迁
- 典籍 / 档案 / 图像 / 碑帖 + 个案 + 问题

## 5. 输出要求

每条建议必须包含：

- 建议类型
- 题名结构模式
- 核心问题
- 拥挤度数值
- 同类题名样本 3-5 条
- 需要补检的文献方向
- 需要避免的表达或论证方式
- 证据强度
