# 栏目结构分析协议

## 1. 定位

栏目结构分析用于判断目标期刊是否存在稳定栏目、临时专题、跨年专题和用户选题的栏目归属。

## 2. 指标

- 栏目首次出现年份
- 栏目最后出现年份
- 连续出现年数
- 出现期数
- 年度覆盖率
- 核心栏目 / 临时栏目

## 3. 判断标准

- 核心栏目：目标年份内每年都出现，或覆盖年份不低于 80%。
- 稳定栏目：连续出现 3 年以上。
- 临时栏目：只出现 1-2 次。
- 专题连续性：同一栏目或专题关键词跨期、跨年重复出现。
