# 给文章润色的交接协议

## 1. 交接定位

只给题名、摘要、关键词、体例和语气约束，不改正文。

## 2. 必填字段

```json
{
  "target_journal": "",
  "title_style": {
    "avg_length": null,
    "subtitle_ratio": null,
    "common_patterns": [],
    "high_frequency_verbs": []
  },
  "abstract_keywords": {
    "abstract_length_range": "",
    "keyword_count_range": "",
    "keyword_order_habits": []
  },
  "format_constraints": [],
  "tone_constraints": [],
  "avoid_patterns": []
}
```

## 3. 必须说明

- 题名结构建议只给模式，不生成正文式题名。
- 摘要和关键词只给长度、数量、顺序习惯。
- 结论尺度和术语倾向必须有证据来源。
