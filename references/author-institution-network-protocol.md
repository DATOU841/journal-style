# 作者机构网络分析协议

## 定位

作者机构网络用于判断目标期刊的作者来源、机构集中度、合作结构和潜在投稿生态。它只服务期刊画像和投稿匹配，不用于推断具体作者关系、私人关系或未公开合作网络。

## 输入

优先使用题录库或选题专项库中的公开元数据：

- `title` / `题名`
- `year` / `年份`
- `author` / `authors` / `作者`
- `institution` / `affiliation` / `单位`
- `column` / `栏目`
- `topic` / `keywords` / `关键词`

作者、机构字段可用 `;`、`；`、`,`、`，`、`、` 分隔。无法确认的机构写入 `unknown`，不得自行补全。

## 指标

- 作者总数、机构总数、单作者/多作者比例。
- 高频作者和高频机构。
- 作者集中度：前 10 作者署名次数 / 总署名次数。
- 机构集中度：机构 HHI，按发文条目计。
- 合作边数量：同一题录内作者两两组合。
- 机构合作边数量：同一题录内机构两两组合。
- 跨机构合作比例：包含 2 个及以上不同机构的题录比例。
- 年度新作者比例：当年首次出现作者 / 当年作者数。

## 判断规则

- 样本低于 50 条：只能写“初步观察”。
- 作者字段缺失率高于 30%：不得给作者集中度高置信判断。
- 机构字段缺失率高于 30%：不得给机构生态高置信判断。
- 跨机构合作比例高但 PDF/RAG 不足时，只能说明题录层合作结构，不能推断实际学术共同体。
- 高频作者/机构只能作为公开题录统计，不写成投稿关系或录用倾向。

## 输出

- `03-analysis/journal-author-institution-network-report.md`
- `03-analysis/journal-author-institution-network-statistics.json`

报告必须包含：

- 样本和缺失率。
- 高频作者和机构。
- 合作边摘要。
- 年度新作者比例。
- 证据强度和降级说明。
- 投稿匹配提示：是否需要补充同机构、同作者、相邻作者群体的检索入库。
