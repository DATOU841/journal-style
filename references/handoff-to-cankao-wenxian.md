# 给参考文献补注的交接协议

## 1. 交接定位

只交接参考文献生态缺口和补注方向，不直接伪造文献。

## 2. 必填字段

```json
{
  "target_journal": "",
  "reference_ecology": {
    "avg_reference_count": null,
    "recent_literature_ratio": null,
    "foreign_literature_ratio": null,
    "self_journal_citation_ratio": null,
    "high_frequency_authors": [],
    "high_frequency_references": []
  },
  "user_reference_gaps": [],
  "recommended_supplement_searches": [],
  "evidence_paths": []
}
```

## 3. 必须说明

- 高频作者和高频文献必须来自可追溯样本。
- 外文文献真实性要单独标注。
- 期刊内互引建议不得变成机械堆引用。
